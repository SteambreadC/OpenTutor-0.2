# OpenTutor :D

📦 一个基于LLM的人工智能学习助手的beta版，功能聚焦于根据各种课程材料，笔记，作业，往年考试题目及例题模拟出考试题目，为教学人员提供帮助！

前端使用 [Vue](https://github.com/vuejs/vue)，后端使用微框架 [Flask](https://github.com/pallets/flask)。
